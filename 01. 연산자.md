### nil 연산자
- a ?? b 
- 옵셔널 a를 벗겨서(unwraps) 만약 a가 nil 인 경우 b를 반환
- a != nil ? a! : b

### 범위 연산자
```swift
// 1 <= index <= 5
for index in 1...5 {  
  print("\(index) times 5 is \(index * 5)")
}
```
```swift
// 0<= i < count
for i in 0..<count {
  print("Person \(i + 1) is called \(names[i])")
}
```
```swift
// 2<=
for name in names[2...] {
  print(name)
}
```
```swift
// <2
for name in names[..<2] {
  print(name)
}
```
```swift
// let 0<= range <= 5
let range = ...5
```
